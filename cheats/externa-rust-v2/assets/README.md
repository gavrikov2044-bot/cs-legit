# Driver Assets

Положи сюда драйверы для встраивания в .exe:

## Файлы:
- `laithdriver.sys` - Наш драйвер (собери из laith-km-driver или возьми из CI/CD)
- `iqvw64e.sys` - Intel драйвер (скачай с https://github.com/TheCruZ/kdmapper/releases)

## Сборка с встроенными драйверами:
```bash
cargo build --release --features embed_drivers
```

## Сборка без драйверов (используются внешние файлы или syscall):
```bash
cargo build --release
```

## Схема работы:

```
┌─────────────────────────────────────────────────────────────┐
│                    ОДИН .EXE ФАЙЛ                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Драйверы встроены?                                         │
│  ├── ДА  → Извлечь в %TEMP% → Загрузить → Удалить          │
│  └── НЕТ → Искать рядом с .exe                             │
│                                                             │
│  Драйвер загружен?                                          │
│  ├── ДА  → Kernel Mode (10x быстрее) 🚀                    │
│  └── НЕТ → Syscall Mode (обходит хуки) ⚡                  │
│                                                             │
│  Syscall работает?                                          │
│  ├── ДА  → Direct NtReadVirtualMemory                      │
│  └── НЕТ → WinAPI ReadProcessMemory (fallback)             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
